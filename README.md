# Репозиторий-портфолио для моих DA проектов
Рекомендую смотреть проекты через NBViwer (https://nbviewer.jupyter.org/github/VladislavZlotsky/data_analysis_projects/tree/main/), т.к. гитхаб не рендерит оглавления проектов, а также интерактивные графики (которых у меня большинство), а также гитхаб немного страшненький.

О проектах:
- проекты с 6-ти месячного курса Профессия аналитик данных. Яндекс.Практикум;
- нумерация проектов (project1, project2...) идет в хронологическом порядке прохождения курса;
- в каждой папке проекта приведем readme.md, в котором приведено краткое описание данных, цели исследования и инструментов.
- примерные времязатраты на выполнение всех 9 проектов около 180 часов. Затраты на изучение теории (тренажер ЯП, теория и доп. материалы) еще около 60 часов.
